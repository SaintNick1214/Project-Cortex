## Key Concepts

- **Auth Context** — Pass user identity and tenant information to all Cortex operations
- **Tenant Isolation** — Complete data separation for multi-tenant SaaS applications
- **Memory Space Isolation** — Fundamental data boundaries (Hive or Collaboration Mode)
- **User Data Isolation** — GDPR-compliant cascade deletion
- **Participant Tracking** — Attribution for Hive Mode debugging
