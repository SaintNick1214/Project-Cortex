## Key Discovery

**`CONVEX_AGENT_MODE=anonymous`** enables headless Convex deployment:
```bash
CONVEX_AGENT_MODE=anonymous npx convex dev --local --once --until-success
```

This was the breakthrough that enabled fully automated local setup.
